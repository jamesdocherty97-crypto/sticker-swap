# Production Artwork

This folder contains app-facing artwork generated from the selected source masters in `public/assets/final-art/selected/`.

- Keep source PNGs intact for traceability.
- Use WebP files here for real UI surfaces.
- Replace artwork by adding a new selected master first, then regenerating this folder and the PWA icons from that master.
- Keep visible product copy in HTML/CSS. Do not bake readable UI text into artwork.

Do not add official tournament, club, national association, publisher, sponsor, kit, badge, flag, official ball, trophy, copied checklist, real player likeness, or readable generated text assets.
