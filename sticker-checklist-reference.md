# Sticker Checklist Reference

The working sticker checklist is embedded in `index.html`.

For the public MVP:

- Base sticker count is `980`.
- Sticker IDs are stable and must not be reordered.
- Promo extras are deferred.
- Display names are used only to help collectors find stickers.

If the checklist is replaced later, keep existing database IDs stable or provide an explicit migration.
